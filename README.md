# Uttrekk
Utrekk av data fra Familia og Visma Flyt Barnevervakt som lagrer dataene i JSON-filer og dokumenter i ZIP-filer i en folderstruktur.

Familia er her delt opp i 15 ulike instanser og databaser. Sokrates inneholder oversikt over alle flyttinger av saker mellom de ulike Familia-instansene.
